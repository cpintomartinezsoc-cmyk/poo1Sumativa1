📘 Descripción general del sistema

Este proyecto corresponde a la actividad Sumativa 1 asignatura Desarrollo Orientado a Objetos I, desarrollada en lenguaje Java utilizando el paradigma de programación orientada a objetos.
El sistema identifica y expresa la información de las personas relacionadas con la empresa turística Llanquihue Tour, permitiendo mostrar clientes, empleados y otros colaboradores mediante una estructura organizada de clases.

🧱 Estructura general del proyecto

src/

├── app/  # Clase principal con el método main

│   └── Main.java  

│
└── model/ # Clases de dominio (Persona, Empleado, Dirección, etc.)
   
    ├── Persona.java
   
    ├── Empleado.java
   
    └── Direccion.java

⚙️ Instrucciones para ejecutar el proyecto

Clonar el repositorio desde GitHub: https://github.com/cpintomartinezsoc-cmyk/poo1Sumativa1.git

Ejecutar la clase:
